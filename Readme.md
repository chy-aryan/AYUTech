A simple frontend web app to explore the idea of integrating artificial intelligence with traditional herbal medicine to offer new and exciting solutions for the future.

## Overview

AYUTech is a web app designed on the idea of how traditional medicine and technology can be integrated.  
This project is currently only designed to work on the **front-end interface** and offers various features that could be included in such an app.

## Features

- Landing page with the introduction of the app  
- Physical wellness section with yoga  
- AI diagnostics interface page  
- Knowledge section with information on herbal medicine  
- E-commerce section with herbal medicine  
- Doctor consultation interface  
- Simple and responsive user interface  

## Tech Stack

- *HTML* – Structure  
- *CSS* – Style  
- *JavaScript* – Basic interaction
